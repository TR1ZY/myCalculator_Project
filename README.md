A simple calculator, coded in Java.
Users have the option of using standard mode for simple operations or scientific for trigonometry operations in radians only.
For simple operations, the user is prompted for an operation first, then they are asked how many numbers they want to use, this feature is also present in scientific mode, but it doesn't apply to trigonometric operations.
For trigonometric operations, only one number is allowed to be input per operation.
After a calculation is done, the user is prompted if they want to do another operation or quit the program.
